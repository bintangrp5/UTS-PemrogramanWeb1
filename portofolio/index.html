Project

Point Of Sales

Poin Of Sales App merupakan aplikasi yang membantu sistem kasir, admin, dan owner, di dalam aplikasi tersebut menyediakan fitur seperti tambah user, tambah produk, laporan transaksi, dan berbagai fitur lainnya, aplikasi yang saya gunakan yaitu Apache Netbeans, dengan database yang saya buat di mysql.

Game Design Document Menteri Kereta : Manajemen Perjalanan Kereta Api

Menteri Kereta: Manajemen Perjalanan Kereta Api adalah sebuah game simulasi yang berfokus pada manajemen perjalanan kereta api, menawarkan pengalaman bermain yang edukatif sekaligus rekreatif. Dalam game ini, pemain dapat mengatur berbagai aspek operasional kereta api, mulai dari penjadwalan hingga pengelolaan rute dan penumpang. Dengan tema transportasi, game ini menyajikan tantangan yang realistis, memungkinkan pemain untuk merasakan bagaimana mengelola sistem transportasi kereta api yang efisien dan tepat waktu.
